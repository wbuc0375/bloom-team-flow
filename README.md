This project can be run locally using Node.js and npm. First, ensure that Node.js (version 18 or newer) is installed on your system, which can be checked by running `node -v` and `npm -v` in your terminal. Once confirmed, download or unzip the project folder and open it in a terminal. Navigate into the project directory (e.g. `cd bloom-team-flow-main`) and install all required dependencies by running `npm install`.

After installation is complete, start the development server by running `npm run dev`. This will launch the application locally, usually at `http://localhost:5173`, which can be opened in a web browser. If that port is already in use, the terminal will provide an alternative local URL.

No additional setup such as databases or environment variables is required. If any issues occur, deleting the `node_modules` folder and running `npm install` again will usually resolve them.
